## 简单工厂模式

现在我们要实现一个简单的商家收银系统，根据商品原价与商家的折扣计算出最终顾客所要支付的金额，最简单的我们会这样实现
```javascript
function cashIn(money,type){
    let result = money;
    switch(type) {
        case '正常收费':
            result = money;
            break;
        case '打五折' :
            result = money/2;
            break;
    }
    return result;
}

```
