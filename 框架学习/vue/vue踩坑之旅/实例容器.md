## 容器的一些特性

### 容器会消失

我们先看一个栗子
```html
<div id="app"></div>
<script >
    var vm = new Vue({
        el: '#app',
        template: '<div>hello world</div>'
    });
</script>
```
也许会有人觉得会渲染成这个样子
```html
<div id="app"><div>hello world</div></div>
```
然而事实是
```html
<div>hello world</div>
```
这个坑踩得很惨···一直以为其他的逻辑将容器这层消除了，最后才发现是这么回事

当你在根实例中用了`template`这个选项，就会用它的DOM结构替换掉容器
