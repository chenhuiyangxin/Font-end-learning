### 对象迭代

```html
<div v-for="(value, key, index) in object">
  {{ index }}. {{ key }} : {{ value }}
</div>
```
简洁的语法

### `v-for 与 v-if`

`v-for`优先级更高，所以`v-if`将会作用于每个循环中，非常有利于我们按照某个条件渲染
